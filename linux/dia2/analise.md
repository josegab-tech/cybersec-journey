# Day 2 - Linux Permissions

## Objective
Understand how file permissions work in Linux.

## Commands Used
ls -l
chmod 700 auth.log
chmod 600 auth.log
chmod 777 auth.log

## Tests Performed
Changed file permissions to observe different access levels.

## Results
It was possible to control who can access the files and how.

## Learning
File permissions are essential for system security.
Avoid using 777 because it allows full access to everyone.

## Notes
This concept is widely used in real-world systems to protect sensitive data.
